## 💻 รายละเอียดและการทำงานของแต่ละแล็บ

### 📝 Lab 2.1: Student Information System
โปรแกรมสำหรับรับข้อมูลนักศึกษาประกอบไปด้วย รหัสนักศึกษา, ชื่อ, อายุ และ GPA จากนั้นจะทำการบันทึกและแสดงผลข้อมูลทั้งหมดออกมา

* **คุณสมบัติ:** มีการสร้าง Method แยกสำหรับการรับค่าแต่ละประเภท (`int`, `String`, `double`) เพื่อความเป็นระเบียบของโค้ด

#### ตัวอย่างการทำงาน (Example Output)
```text
Enter Id: 65010001
Enter Name: Somchai
Enter Age: 20
Enter GPA: 3.50
```
### Lab 2.2: Grade Calculator
โปรแกรมรับคะแนนสอบจากผู้ใช้งานเพื่อนำมาคำนวณและตัดเกรดตามเงื่อนไข (If-Else Condition)
เกณฑ์การตัดเกรด:
* คะแนน $\ge$ 80 ได้เกรด A
* คะแนน $\ge$ 70 ได้เกรด B
* คะแนน $\ge$ 60 ได้เกรด C
* คะแนน $\ge$ 50 ได้เกรด D
* คะแนน $<$ 50 ได้เกรด F

###  ตัวอย่างการทำงาน (Example Output)
```
GPA: 75
Grade:  B
```
### ✖️ Lab 2.3: Multiplication Table
โปรแกรมแสดงสูตรคูณแม่ 2 ตั้งแต่ $2 \times 1$ ไปจนถึง $2 \times 12$ โดยการประยุกต์ใช้คำสั่งวนซ้ำ (for loop)
### ตัวอย่างการทำงาน (Example Output)
```
2*1=2
2*2=4
...
2*12=24
